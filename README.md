# 🎮 BigodeStore

BigodeStore é uma landing page moderna para uma loja digital de jogos, inspirada na experiência de plataformas como Xbox Store e Steam. O projeto foi desenvolvido utilizando apenas HTML e CSS, apresentando uma interface elegante, responsiva e focada na divulgação e venda de jogos digitais.

---

## 🚀 Funcionalidades

- Banner principal com destaque para jogo em promoção.
- Navegação simples e intuitiva.
- Seção de benefícios da plataforma.
- Catálogo de gêneros de jogos.
- Área de jogos mais vendidos.
- Lista de jogos populares.
- Seção de pré-venda (Coming Soon).
- Avaliações de clientes.
- Rodapé com links úteis e redes sociais.

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- Google Fonts (Lexend)

## 📸 Seções da Página

### 🏠 Hero Section
Destaque principal da loja com promoção do jogo MultiVersus e botão de chamada para ação.

### ⭐ Why BigodeStore
Apresenta os diferenciais da plataforma:

- Mais de 5000 jogos disponíveis
- Mais de 100 mil clientes satisfeitos
- Compatibilidade com consoles Xbox

### 🎮 Games Genres
Catálogo visual de categorias:

- Strategy
- FPS
- RPG
- Puzzle
- Role Playing
- Sports

### 🔥 Best Seller Games
Lista dos jogos mais vendidos da plataforma.

### 📈 Popular Games
Exibição dos jogos mais populares com promoções e downloads gratuitos.

### 🚀 Coming Soon
Área dedicada aos próximos lançamentos disponíveis para pré-venda.

### 💬 Gamers Satisfied
Avaliações de clientes para aumentar a confiança dos usuários.

### 📱 Footer
Rodapé com:

- Menu de navegação
- Recursos úteis
- Redes sociais
- Informações da empresa
